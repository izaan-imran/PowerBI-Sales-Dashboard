# 📊 Superstore Sales Performance Analysis

A comprehensive Data Science project focusing on business health, regional trends, and statistical insights using Power BI and analytical modeling.

## 🎯 Project Objective
The goal of this project is to analyze the 'Superstore' dataset to identify key drivers of sales and profit. I focused on building a dashboard that not only shows totals but also reveals the statistical distribution of performance across different regions and categories.

## 📂 Repository Contents
- **[Interactive Dashboard](./sales-performance-dashboard.pbix):** The original Power BI file for full interactivity.
- **[Data Dataset](./superstore-sales-data.csv.csv):** The cleaned source data used for the analysis.
- **[PDF Report](./sales-performance-dashboard.pdf):** A static version for quick viewing without Power BI software.
- **[Dashboard Preview](./sales-performance-dashboard.jpg):** High-resolution screenshot of the final visualization.

## 🛠️ Technical Process & Statistics
- **Data Cleaning:** Used Power Query to handle missing values and ensure data integrity for accurate reporting.
- **Data Modeling:** Applied database normalization principles to ensure a structured data flow.
- **Measures & DAX:** Created custom measures for Profit Margin %, Total Quantity, and Sales metrics.
- **Statistical Focus:** Analyzed the relationship between sales volume and profitability across different product categories (Furniture, Office Supplies, and Technology).

## 💡 Key Insights
- **Profitability:** The dashboard reveals a Profit Margin of 0.27%, indicating a high-volume but low-margin business model.
- **Regional Performance:** The map visualization identifies the West region as a primary driver of sales.
- **Category Trends:** Office Supplies represent a significant portion of the total quantity sold (4,566 units), whereas Technology often drives higher individual transaction value.
